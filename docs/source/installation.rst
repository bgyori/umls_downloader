Installation
============
The most recent release can be installed from
`PyPI <https://pypi.org/project/umls_downloader>`_ with:

.. code-block:: shell

    $ pip install umls_downloader

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    $ pip install git+https://github.com/cthoyt/umls_downloader.git

To install in development mode, use the following:

.. code-block:: shell

    $ git clone git+https://github.com/cthoyt/umls_downloader.git
    $ cd umls_downloader
    $ pip install -e .
